Tauon Music Box

A desktop music player

Features:

  • Unintuitive interface
  • Ugly rudimentary design
  • Missing common playback features like Replay-Gain and gapless playback
  • Poorly optimized with excessive RAM and CPU usage
  • Very limited customization options
  • Random bugs and glitches

You're still here? Here's some of the more positive features:

  • Dark themed interface
  • Simple drag and drop track importing
  • Batch transcode folders of tracks to OGG or OPUS for easy compressing and copying to device (requires ffmpeg)
  • Automatic CUE sheet detection and integration
  • Quick search
  • Create sorted playlists
  • Last.fm scrobbling
  • Outbound streaming (requires icecast)
  • Multiple layout views including 'large album art' view and gallery view
