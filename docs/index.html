Tauon Music Box

A desktop music player

Features:

  • Playback support for most common codecs including MP3, FLAC, OGG, OPUS, and APE
  • Simple drag and drop track importing
  • Dark themed interface
  • Batch transcode folders of tracks to OGG or OPUS to save space when copying to portable device (requires ffmpeg)
  • Automatic CUE sheet detection and integration
  • Quick search
  • Multiple playlist sorting and filtering presets
  • Last.fm scrobbling
  • Outbound streaming via Icecast
  • Multiple layouts including large album art and gallery view
