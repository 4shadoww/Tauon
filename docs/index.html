Tauon Music Box

A desktop music player

Features:

  • Simple drag and drop track importing
  • Dark themed interface
  • Batch transcode folders of tracks for easy compressing and copying to device (requires ffmpeg)
  • Automatic CUE sheet detection and integration
  • Quick search
  • Create sorted playlists
  • Last.fm scrobbling
  • Outbound streaming via Icecast
  • Multiple layout views including large album art and gallery view
