Tauon Music Box

An alternative desktop music player

Features:

  • Playback support for MP3 and FLAC, plus others.
  • No file pickers or 'libraries' just simple drag and drop importing.
  • Automatically handles CUE sheets.
  • Streamlined quick searching.
  • Transcode folders to OGG or OPUS for saving space on DAP's.
  • Outbound radio streaming via HTTP.
  • Inbound radio streaming with recording and splitting.
  • Integrated gallery view.
  • MPRIS interface support.
  • Uses the proprietary BASS audio library.
